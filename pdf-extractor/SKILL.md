# pdf-extractor: PDF 數據提取專家

把那種死板的 PDF 報表直接轉成可以運算的表格或摘要。

## 核心指令
- `/pdf-to-table [PDF 內容]`：將掃描或死板的表格內容轉換為 Markdown、CSV 或 Excel 格式。
- `/pdf-extract-summary [PDF 內容]`：提取長篇 PDF 文件中的關鍵數據點與摘要。
- `/pdf-compare [PDF A] [PDF B]`：對比兩份 PDF 文件的關鍵數據差異。

## 使用場景
- 處理 PDF 格式的財報、產業分析報告或合約。
- 快速獲取長篇專業文件的數據洞察。
