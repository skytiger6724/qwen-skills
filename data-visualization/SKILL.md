# Data Visualization Skill

Focus on "Insights over Graphics". This skill ensures every chart provides clear business value.

## Workflow
1. **Data Cleaning**: Check for missing values or outliers.
2. **Chart Selection**: Use line charts for trends, bar charts for comparison, and scatter plots for correlation.
3. **Aesthetic Standards**: Use consistent color schemes and clear labels.
4. **Insight Generation**: Provide at least 3 key business insights for every visualization.

## Technical Stack
- **Python**: Matplotlib, Seaborn, Plotly.
- **Web**: D3.js, ECharts, Chart.js.
